To start the application using Docker please use build-and-run.docker.ps1 script attached to this project.
This will utilise Docker Compose to orchestrate the application startup and configuration.

The application code contains certificates and private keys to establish secure connections over TLS. The presence of private keys is safe in this case because these are for test purposes only.
